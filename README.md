## 🚀 開発の始め方

### 必要な環境

- Bun (パッケージマネージャー)

### 開発環境のセットアップ

```bash
# 依存関係のインストール
bun install

# 開発サーバーの起動
bun run dev

# 型チェックの実行
bun run typecheck
```

### コード品質管理

```bash
# コードフォーマット
bun run format

# リントチェック
bun run lint
```

### UIコンポーネント開発

```bash
# Storybookの起動
bun run storybook

# SVGアイコンのコンポーネント生成
bun run icongen
```

```bash
# プレビュー環境へのデプロイ
bun run deploy
```
